## Prerequisites

- Install a current Node.js LTS release
- Install the Mintlify CLI
- Work from the docs root where `docs.json` lives

## Local preview

```bash
npm i -g mint
mint dev
```

Mintlify will start a local preview, usually on `http://localhost:3000`.

## Recommended working loop

- Confirm the source of truth in the product code before you write
- Update the relevant MDX file and `docs.json` when navigation changes
- Preview the changed pages locally
- Run a broken link check before you ship

## Broken link check

```bash
mint broken-links
```
