Use Claude Code when you want help drafting or refactoring docs, but keep the application code as the source of truth.

## Recommended setup

- Open the docs repo root so `AGENTS.md` is loaded
- Keep the backend and frontend paths available for verification:
  - `/Users/yasseen/Documents/brezel.ai/BrezelScraper/brezelscraper-backend`
  - `/Users/yasseen/Documents/brezel.ai/BrezelScraper/brezelscraper-frontend`
- Install the Mintlify skill if you want component help:

```bash
npx skills add https://mintlify.com/docs
```

## Working rules

- Ask Claude Code to verify live behavior in backend handlers before rewriting API pages
- Prefer concrete tasks such as "document the results pagination response" over vague prompts
- Review terminology against `AGENTS.md` before accepting broad rewrites
