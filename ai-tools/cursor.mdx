</Note>

Use Cursor when you want fast inline editing with the docs, backend, and frontend open side by side.

## Recommended setup

- Open the docs repo root so `docs.json` and `AGENTS.md` are in scope
- Keep the backend and frontend codebases open when documenting product behavior
- Install the Mintlify skill if you want component help:

```bash
npx skills add https://mintlify.com/docs
```

## Working rules

- Verify API payloads against Go handlers and model structs
- Verify dashboard steps against the frontend before publishing UI instructions
- Remove or rewrite starter content instead of hiding it
