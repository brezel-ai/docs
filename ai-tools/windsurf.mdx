Use Windsurf when you want AI-assisted editing with fast access to the docs and application code together.

## Recommended setup

- Open the docs repo root first
- Keep these source-of-truth codebases available:
  - `/Users/yasseen/Documents/brezel.ai/BrezelScraper/brezelscraper-backend`
  - `/Users/yasseen/Documents/brezel.ai/BrezelScraper/brezelscraper-frontend`
- Install the Mintlify skill if you want component help:

```bash
npx skills add https://mintlify.com/docs
```

## Working rules

- Treat backend handlers and frontend flows as authoritative
- Keep prompts specific to one workflow or endpoint at a time
- Review final wording for BrezelScraper terminology before merge
